## Getting started

- Clone the repo: `git clone https://github.com/koders-in/X10-Superlist-clone.git`
- Install with [yarn](https://yarnpkg.com/): `yarn install`
- Run with [yarn](https://yarnpkg.com/): `yarn start`

## Running

### Built with

1. Run `yarn install` to install the Node.js dependencies.
2. Run `npm start` to run the website on a localhost
3. Open `http://localhost:3000/` in your browser, and voilà.
